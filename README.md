# 🎵 İsimden Melodi (Name Melody Generator)

Bu proje, girilen ismi harflere ayırarak her harfi bir frekansa çeviren ve bunları melodik bir yapıda çalan bir **Web Audio API tabanlı müzik üretici uygulamasıdır.**

---

## 🚀 Özellikler

* 🧠 İsmi harflere ayırıp nota/frekansa çevirme
* 🎼 Otomatik melodi üretimi
* 🎭 3 farklı müzik tarzı:

  * Mutlu (Happy)
  * Sakin (Calm)
  * Epik (Epic)
* 🔁 Tekrar eden harflerde farklı oktav sistemi
* 🎨 Koyu tema + açık tema desteği
* 🟡 Gizli “Birlik Tema” modu (sarı-siyah özel tema)
* 🔒 Tema değiştirme kilitleme sistemi
* 📱 Mobil + PC uyumlu responsive tasarım
* 🎧 Web Audio API ile gerçek zamanlı ses üretimi

---

## 🧩 Nasıl Çalışır?

1. Kullanıcı isim girer
2. Her harf bir frekansa eşlenir
3. Seçilen stile göre:

   * tempo
   * dalga tipi
   * oktav değişimi uygulanır
4. AudioContext ile ses üretilir
5. OfflineAudioContext ile melodi render edilir

## 🎛️ Tema Sistemi

### 🌙 Normal Tema
